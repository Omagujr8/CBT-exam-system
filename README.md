# CBT Exam System

## Overview
This project is a Computer-Based Testing (CBT) system designed to manage online examinations securely and efficiently. It supports exam creation, candidate participation, automated scoring, and result management.

## Motivation
The project was built to address challenges in exam integrity, result processing, and accessibility in educational institutions.

## Features
- User authentication (admin & candidates)
- Exam creation and scheduling
- Timed examinations
- Automated grading
- Result storage and retrieval
